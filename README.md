# Ground Engineering RAG Assistant

A Streamlit-based document search and question-answering application for ground engineering reports.

## Features
- Search across project documents
- PDF and DOCX text extraction
- FAISS vector search
- SentenceTransformer embeddings
- Mistral API integration
- Groq fallback support
- Source document previews

## Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- SentenceTransformers
- PyPDF2
- python-docx

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
